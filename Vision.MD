# The vision for the nixos-config

1. KISS: Keep it simple and straightforward.
2. Nixos is a basis: Nix shall not step out of its bounds. When possible, use what the program expects
3. Dentric: Each module should be independant, self configured. Read https://github.com/mightyiam/dendritic
4. Optimized: Evaluation time should be reduced
5. Fast Feedback Loop: System modifications go through Nix/Nixos. Configuration modifications should be made as fast as possible.
